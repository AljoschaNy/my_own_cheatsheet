Table <table>

Tables are used to arrange data into rows (tr) and columns.
To give the table a semantically structure thead, tbody and tfoot were introduced.
Whereas thead is used to declare the headings for each column, tbody is used to fill the cells corresponding to the headings. In the tfoot you can sum up a column for example.

Sample: 
                                    
                                        <table>
                                            <thead>
                                                <tr>                               
                                                    <th scope="col">Header1</th>   i. scope tells whether header th is a header for a column or row 
                                                </tr>
                                            <thead>
                                            <tbody>
                                                <tr>
                                                    <td>Cell1</td>
                                                </tr>
                                            <tbody>
                                            <tfoot>
                                                <tr>
                                                    <td>Sum1</td>
                                                </tr>
                                            <tfoot>
                                        </table>
